    Critérios de Aceitação

    US001 - Login

         A aplicação deve mostrar uma tela de boas-vindas com opções para "Começar agora" e "Já possuo conta".
         O usuário deve poder clicar no botão "Já possuo conta".
         O sistema deve exibir uma tela de login com campos para inserir o e-mail/nickname e a senha, esqueci a senha, login com Facebook, login com Google e login com a Apple.
         O usuário deve inserir seu e-mail/usuário e senha nos campos apropriados.
         O sistema deve verificar as credenciais fornecidas.
         O sistema deve autenticar o usuário e redirecionar para a tela principal do Duolingo.

    US002 - Visualizar Perfil

         O usuário deve poder acessar a tela de perfil a partir da tela principal.
         A tela de perfil deve exibir as informações do usuário:
             Foto de perfil ou avatar;
             Nome;
             Nickname;
             Tempo de inscrição;
             Cursos escolhidos;
             Quantidade de amigos que o usuário segue;
             Quantidade de seguidores do usuário;
             Ofensiva atual;
             Total de XP acumulado;
             Divisão atual;
             Quantidade de pódios;
             Conquistas desbloqueadas;

         Deve existir um botão de adicionar amigo.
         Deve existir um botão para compartilhar o perfil do usuário.
         O sistema deve mostrar sugestões de outros usuários.

    US003 - Resolver Exercício do Curso Escolhido

         O usuário deve poder acessar a trilha de exercícios da unidade na tela principal.
         O sistema deve exibir as lições disponíveis para o usuário.
         O usuário deve poder selecionar a lição que deseja praticar.
         O sistema deve exibir o exercício escolhido.
         O sistema deve mostrar a questão e indicar a forma adequada de resposta (digitando, selecionando ou falando).
         O usuário deve resolver o exercício.
         O sistema deve verificar a resposta do usuário.
         O sistema deve dar feedback sobre a resposta do usuário.
         O sistema deve mostrar tela de conclusão do exercício.
         O sistema deve atribuir XP ao usuário ao concluir o exercício.
         O sistema deve avançar para a próxima lição ou unidade após a conclusão do exercício.

    US004 - Acessar Conta em Dispositivos Diferentes

         O usuário deve poder realizar o login na plataforma em diferentes dispositivos.
         O sistema deve manter o usuário autenticado sem a necessidade de login a cada acesso em múltiplos dispositivos simultaneamente.
         O progresso do usuário deve ser sincronizado em tempo real ao alternar entre dispositivos ativos.
         Em casos de conflitos de progresso (e.g., um usuário faz progresso em dois dispositivos desconectados), o sistema deve adotar uma lógica para resolver a inconsistência, como priorizar o progresso mais recente

    US005 - Alterar Curso

         O usuário deve poder selecionar o botão para alterar o curso na tela principal.
         O usuário deve poder visualizar a lista dos cursos já adicionados por ele.
         O sistema deve exibir um botão de adicionar novo curso.
         O sistema deve exibir uma lista de cursos disponíveis para adicionar.
         O usuário deve poder selecionar o curso que deseja aprender.
         O sistema deve alterar o idioma do usuário na plataforma.
         O sistema deve exibir uma mensagem de confirmação da alteração do idioma.
         O sistema deve redirecionar o usuário para a tela principal após a alteração do idioma.
         O sistema deve exibir o idioma selecionado na tela principal.

    US006 - Revisar Exercícios Errados

         O usuário deve poder acessar uma seção dedicada para revisar exercícios a partir do menu principal.
         O usuário deve poder acessar a tela de revisão de exercícios errados.
         O sistema deve exibir a lista de exercícios errados.
         O usuário deve poder começar a lição de revisão de exercícios respondidos errados.
         O sistema deve exibir o exercício.
         O usuário deve poder resolver o exercício.
         O sistema deve verificar a resposta do usuário.
         O sistema deve dar feedback sobre a resposta do usuário.
         O sistema deve mostrar a tela de conclusão do exercício.
         O sistema deve atribuir XP ao usuário ao concluir o exercício.

    US007 - Visualizar Progresso

         O usuário deve poder acessar seu progresso na tela de progresso.
         O sistema deve exibir o progresso do usuário em XP e nível.
         O usuário deve poder visualizar o progresso de cada habilidade ou módulo, incluindo o nível de conclusão e as lições restantes.
         O sistema deve mostrar as habilidades que precisam de revisão (baseado no tempo desde a última prática) devem ser destacadas.

    US008 - Compartilhar Progresso

         O sistema deve exibir um botão de compartilhamento ao concluir uma lição.
         O sistema deve exibir um botão de fechar componente de compartilhamento.
         O sistema deve mostrar ao usuário as opções de redes sociais para o compartilhar seu progresso.
         O usuário dever ser capaz de clicar na opção de uma rede social.
         O sistema deve abrir a rede social selecionada pelo usuário.
         O sistema deve gerar uma imagem do progresso para o usuário compartilhar.
         O sistema deve exibir uma mensagem de confirmação do compartilhamento.

    US009 - Visualizar meu ranking

         O usuário deve poder acessar sua posição no ranking a partir do menu.
         O sistema deve exibir a posição do usuário no ranking.
         O sistema deve exibir a quantidade de XP do usuário.
         O sistema deve exibir a quantidade de XP dos outros usuários da liga.
         O sistema deve exibir ordenar os usuários por quantidade de XP do que possuir maior quantidade de XP para o quer possui menos.

    US010 - Visualizar minha liga

         O usuário deve poder acessar sua liga a partir do menu principal.
         O sistema deve exibir o nome da liga atual do usuário.
         O sistema deve exibir a quantidade de XP do usuário.
         O sistema deve exibir a quantidade de XP dos outros usuários da liga.
         O sistema deve exibir a posição do usuário na liga.
         O sistema deve exibir todos os usuários da liga ordenado-os pelo XP, do que possui a maior quantidade de XP para o que possui a menor quantidade.

    US011 - Compartilhar Liga ou Ranking

         O sistema deve exibir um botão de compartilhamento ao acessar a liga ou ranking.
         O sistema deve exibir um botão de fechar componente de compartilhamento.
         O sistema deve mostrar ao usuário as opções de redes sociais para o compartilhar sua liga ou ranking.
         O usuário dever ser capaz de clicar na opção de uma rede social.
         O sistema deve abrir a rede social selecionada pelo usuário.
         O sistema deve gerar uma imagem da posição da liga ou ranking que o usuário está para ele compartilhe.

    US012 - Visualizar o Perfil de Outro Usuário

         O usuário deve poder acessar o perfil de outro usuário a partir da liga.
         O sistema deve exibir o perfil do usuário visitado.
         Deve existir um botão de seguir.
         Deve existir um botão para compartilhar o perfil do usuário.
         O sistema deve exibir as informações do usuário visitado:
             Foto de perfil ou avatar;
             Nome;
             Nickname;
             Tempo de inscrição;
             Cursos escolhidos;
             Quantidade de amigos que o usuário segue;
             Quantidade de seguidores do usuário;
             Progresso semanal do outro usuário.
             Ofensiva atual;
             Total de XP acumulado;
             Divisão atual;
             Quantidade de pódios;
             Conquistas desbloqueadas;

         Deve existir um botão de reportar usuário.
         Deve existir um botão de bloquear usuário.

    US013 - Adicionar outro usuário

         O usuário deve poder selecionar o botão de adicionar amigos a partir do seu perfil.
         O sistema deve exibir uma lista de maneiras adicionar um amigo, sendo elas:
             Escolher dos contatos;
             Adicionar por nome ou nickname;
             Compartilhar o seu próprio perfil.

         O usuário deve poder adicionar um amigo a partir da lista de sugestões de usuários.
