    Editar Avatar do Perfil


    Título Editar Avatar do Perfil
    Objetivo O usuário pode editar o avatar de seu próprio perfil
    Contexto Pré-condição:
    1. O usuário deve estar logado na aplicação
    2. O usuário deve possuir um avatar
    Pós-condição:
    1. O usuário edita o avatar de seu perfil
    Atores Usuário
    Recursos Acesso à internet
    Episódios 1. O usuário acessa a aplicação
    2. O usuário clica no ícone de mais opções
    3. O usuário seleciona a opção de perfil
    4. O usuário clica no avatar
    5. O usuário escolhe as características do avatar
    6. O usuário salva o avatar
    Restrições Fluxo intuitivo
    Exceções 1. A internet deixa de funcionar
    2. O usuário sai da aplicação durante a ação
